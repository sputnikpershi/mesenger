# Navigator

### Приложение-прототип социальной сети, на пободобие VK.

#### Стэк текхнологий:
 - FireBase
 - CollectionView
 - CoreData
 - Predicate
 - GCD
 
 
 В приложении реализована авторизация через FireBase: тестовый номер телефона (966-966-66-66)  и код (666-666)
